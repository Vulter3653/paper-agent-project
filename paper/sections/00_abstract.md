# Abstract

Paper Agent is an AI Agent-based literature discovery and evaluation system for business-school research workflows. It decomposes scholarly search into a traceable 12-stage pipeline that combines planning, journal selection, retrieval, journal filtering, DOI and metadata verification, open-access lookup, storage, evaluation preparation, relevance assessment, ranking, critic review, and report delivery. The latest evaluation layer is Benchmark v3, a six-layer, 30-metric automated framework covering reproducibility, schema quality, deterministic validity, retrieval accuracy, semantic-quality auditing, and robustness. The promotion gate status is **PASS WITH CLAIM BOUNDARIES**. Layers 1--4 and Layer 6 are computed. Layer 5A is a quota-limited partial implementation audit with 22 of 125 rows evaluated (17.6%); its successful subset contains no Proposed Agent rows. Layer 5B provides a deterministic semantic proxy for all 125 rows, but it does not replace LLM or human semantic evaluation. Baseline comparison is restricted to the T001--T003 common-support subset, while T004--T020 remain artifact-level validation tasks. These results support a reproducible benchmark framework with explicit reporting limits, not full comparative superiority or full semantic-quality validation.
