# 4인 팀 프로젝트 (Team Project)

이 리포지토리는 4인 팀 프로젝트를 위한 공통 작업 공간입니다.

## 폴더 구조 (Directory Structure)

- `integrated/`: 4명의 개별 작업물을 종합하고 통합한 코드가 저장되는 공간입니다.
- `seunghyeon_choi/`: SeunghHyeon Choi 팀원의 개인 작업 공간입니다.
- `shonshinemin_cmd/`: shonshinemin-cmd 팀원의 개인 작업 공간입니다.
- `jin23624_cpu/`: jin23624-cpu 팀원의 개인 작업 공간입니다.
- `juilie_bot_hub/`: juilie-bot-hub 팀원의 개인 작업 공간입니다.

## 협업 규칙

1. 자신의 이름으로 된 폴더에서 자유롭게 개발을 진행합니다.
2. 기능이 완성되거나 통합이 필요한 경우 `integrated/` 폴더로 코드를 옮기고 병합(Merge) 절차를 거칩니다.
3. 커밋 메시지는 명확하게 작성해 주세요.
